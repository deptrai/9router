# Project Instructions

## Language

- Luon tra loi nguoi dung bang tieng Viet.
- Luon viet tai lieu, ghi chu, spec, PRD, architecture document, test plan, changelog, va cac artifact khac bang tieng Viet.
- Giu nguyen thuat ngu ky thuat, ten framework, ten API, ten module, command, file path, bien moi truong, code symbol, va error message bang tieng Anh khi do la cach goi chuan.
- Khi mot khai niem tieng Anh quan trong co the gay mo ho, viet theo dang: tieng Viet truoc, thuat ngu English trong ngoac don. Vi du: "kiem thu hoi quy (regression testing)".
- Khong dich ten rieng cua san pham, thu vien, giao thuc, model, provider, package, hoac service.

## Response Style

- Tra loi ngan gon, ro viec, uu tien hanh dong tiep theo.
- Khi giai thich code hoac architecture, dung tieng Viet de dien giai va giu code identifiers bang tieng Anh.
- Khi tao hoac cap nhat document, khong chuyen sang tieng Anh tru khi nguoi dung yeu cau ro.
